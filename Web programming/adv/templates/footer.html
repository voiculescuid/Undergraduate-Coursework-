About us

Cyber attacks are becoming more diverse and innovative, so we need customized solutions.The solution envisages the protection of the user's files against malware software that has passed the undetected scan stage. The objective of the application is to interfere with the natural flow of the attack and to protect the file system by intercepting and rejecting the system calls that wish to modify the files protected by the user. The file system added to the protection will remain unchanged regardless of the state of the operating system.
